The skins found here require eXastum version 3.x
To install a skin, download it's folder and copy it into the skins folder in your eXastum folder.
Then open eXastum, open the main menu and select 'Installer' and choose 'Skin'.
Type the name of the skin's folder inside the skins folder and make sure the case is corrct as it is case-sensitive.

Example:
To install a theme called 'nova' you would copy it to the folder like so:
eXastum/skins/nova/
Then when installing the skin type 'nova' and you're done.
