Welcome to eXastum



Thank you for downloading eXastum 2.0! Your interest in the project is of great importance. This is setup. Setup will guide you through configuring eXastum to make it work right for you. It will allow you to create an account for yourself and set your default settings. You will get the chance to make eXastum suit your style and your way of working. eXastum is an open source project, you can contribute code if you wish, just send an email to brianmillar9@yahoo.com. You can also send me feedback, suggestions or anything else you want. eXastum is designed to be open, free and fun and will hopefully soon be community driven. If you are seeing this after already running setup please file a bug report.



Next Step

Technical System Setup



If you are not running eXastum from a localhost server please close the browser now, set up your server and then reload eXastum from it. You can continue however the file system and some other features will not work properly. If you do not know how to set up or get a localhost server please search for "XAMPP" on Google. When you are ready to create the file system and setup all the default system settings please click the "Setup System Now" button below.




*The space you give will not be used now but will fill up as you add files to eXastum. Once the limit has been reached file writes will fail.
